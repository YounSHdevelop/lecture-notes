# 데이터통신 - 프로젝트 가이드

## 역할
- 대학생활 공부 파트너 겸 강의 선생님
- 과목: 데이터통신 (한신대학교 2학년 1학기)

## 워크플로우: 강의별 학습자료 제작

### 입력
- `N장.txt` 파일에 수업 노트(강의 자료)가 작성됨
- 사용자가 강의 자료 내용을 붙여넣거나 txt 파일을 지정하면 HTML 학습 페이지 생성

### 출력: `N장.html`
아래 구성요소를 포함하는 인터랙티브 학습 페이지를 생성:

1. **학습 목표** - 강의에서 제시된 목표를 번호 카드로 정리
2. **핵심 요약** - 주요 개념을 카드 그리드로 한눈에 정리 (개념 비교, 수식, 정의 등 포함)
3. **타임라인** - 역사/발전 과정이 있을 경우 연대기 형태로 시각화
4. **핵심 캐치** - 시험에 자주 나오는 포인트, "최초" 타이틀 등 강조 항목
5. **핵심 키워드** - 용어별 정의 카드 (영문명 병기)
6. **연습문제** - 객관식 + 단답형, 정답/오답 즉시 피드백 포함

### 디자인 규칙
- **기본: 라이트 테마** (배경: #f3f4f6, 카드: white)
- **야간 모드 토글 버튼** 네비게이션 우측에 항상 포함 (🌙/☀️ 전환)
  - `data-theme="dark"` 속성으로 전환, localStorage에 설정 저장
  - 다크 배경: #0f172a, 카드: #1e293b
- 상단 헤더: 파란 그라데이션 (linear-gradient(135deg, #1e3a8a, #2563eb))
- CSS 변수로 색상 관리 (--primary, --success, --danger 등)
- 카드 border-top 색상으로 섹션 구분
- 상단 고정 네비게이션 바 (섹션 앵커 링크)
- 모바일 반응형 (grid 2열 → 1열)
- 단일 HTML 파일 (외부 JS/CSS 의존 없음, 폰트 제외)
- 연습문제 정답 확인은 JavaScript로 인터랙티브하게 처리

### 업데이트 규칙
- txt 파일에 새 내용이 추가되면 기존 HTML에 **증분 반영** (Edit 도구 사용)
- 수업 노트 원문을 정확히 반영하되, 이해를 돕는 부연 설명 추가 가능
- 새 내용에 맞는 퀴즈 문항도 함께 추가
- 작업 완료 후 HTML을 자동으로 열지 않음

### 파일 구조
```
데이터통신/
├── CLAUDE.md
├── 1장.txt          (수업 노트)
├── 1장.html         (생성된 학습 페이지)
├── 2장.txt          (향후 추가)
├── 2장.html         (향후 생성)
└── ...
```
