데이터 : 현실 세계에서 단순한 관찰 또는 측정을 통해 수집한 사실이나 값을 문자,숫자,기호 등으로 표현한것
정보 : 데이터를 가공 처리하여 가치와 의미를 부여한것

통신과 데이터 통신
통신 : 두 사람(장치)이상의 정보 공유를 위해 정해진 규칙(프로토코로)에 따라 한 지점에서 다른 지점으로 정보를 전달하는 과정(예시 : 대화, 그림, 글씨, 모닥불 연기등, 각각의 규칙)
데이터통신 : 컴퓨터 같이 통신 기능이 있는 둘 이상의 통신 장치 사이에서 전송매체를 통하여 정해진 규칙에 따라 데이터로 표현된 정보를 교환하는 과정

데이터 통신 = 데이터 처리 + 데이터 전송

데이터 통신 네트워크
지리적으로 분산된 여러 정보원에 정보를 전송하고 이를 공유하기 위해서 설계된 상호 연결 시스템
데이터 통신 네트워크 = 컴퓨터 네트워크 = 네트워크

데이터 통신의 역사
1837년 전신기 모르스가 전신기 개발, 최초의 전기통신
1876년 전화 벨이 전화 발명, 음성통신의 시초
1895년 무선전신 마르코니가 무선전신기 발명
1946년 eniac 에거트와 모클리가 개발한 컴퓨터, 최초의 전자식 컴퓨터
1951년 univac 에커트와 모클리가 유니박을 개발하여 인구조사통계국에 설치, 최초의 상업용 컴퓨터
1958년 sage 미국 공군이 개발한 반자동 반광시스템, 최초의 데이터 통신 시스템 
1963년 sabre 아메리칸 에어라인 항공사에서 도입한 좌석 예약 시스템, 최초의 상업용 데이터 통신 시스템
1964년 ctss mit에서 개발한 최초의 시분할 시스템,대학 내 대형 컴퓨터의 공동 이용 목적 
1969년 arpanet 미국 국방성이 개발한 최초의 패킷 통신망, 인터넷의 효시
1970년 aloha 미국 하와이 대학에서 개발한 최초의 무선 패킷 통신망
1974년 sna ibm사가 개발한 컴퓨터 간 접속 표준
1975년 telent, tymnet 패킷 교환 방식의 데이터 통신 서비스 개시
1990년 www 최초의 하이퍼텍스트 브라우저 개발 및 하이퍼텍스트 등장

###시험문제 : 데이터 통신 시스템을 구성하는 요소를 적으시오.

단말장치 : 입출력, 전송제어,기억기능
신호변화 장치 : 데이터를 전송에 적합한 신호로 변환하는 장치 -> 모뎀
모뎀 : 디지털데이터를 아날로그 신호로 변환하여 전송
dsu/csu(digital service unit/channel service unit) : 디지털 데이터를 디지털 신호로 변환하여 전송, csu는 여러개의 채널을 다중화하여 전송하는 장비(t-1방식, e-1방식)

통신제어장치 : 대형 컴퓨터가 데이터 처리에만 전념할 수 있도록 대형 컴퓨터를 대신하여 데이터 전송에 관한 전반적인 제어기능을 수행
ccp,fep

통신제어장치의 기능 : 전송제어,동기 및 오류제어, 기타 기능

통신회선 : 데이터가 전달되는 전송매체를 의미, 유선 매체와 무선 매체로 구분
유선 : 트위스티드 페어, 동축케이블, 광케이블
무선 : 전자파, 적외선

네트워크의 필요성
점대점 연결 : 가장 간단한 형태이지만 통신 비용 측면에서 경제적이지 못함.
네트워크를 이용한 연결 : 각 단말 장치는 한 개의 통신 포트만으로 네트워크의 노드에 접속, 네트워크 설비를 공유함으로써 단말 장치 간의 연결에 드는 비용은 크게 줄일 수 있음.
