핵심 요약

데이터 vs 정보

  • 데이터 : 관찰·측정으로 수집한 사실을 문자·숫자·기호로 표현
  • 정보 : 데이터를 가공·처리하여 가치와 의미를 부여한 것

통신 vs 데이터 통신

  • 통신 : 두 장치 이상이 프로토콜에 따라 정보를 전달하는 과정
  • 데이터 통신 : 통신 기능이 있는 장치들이 전송매체를 통해 데이터를 교환하는 과정

데이터 통신 = 처리 + 전송

데이터 통신 = 데이터 처리 + 데이터 전송

컴퓨터(처리)와 통신(전송)의 결합으로 탄생

데이터 통신 네트워크

  • 지리적으로 분산된 정보원을 상호 연결한 시스템
  • 컴퓨터 네트워크 = 데이터 통신 네트워크 = 네트워크 (모두 동일 의미)

데이터 통신 시스템의 5가지 구성요소 ⚠️ 시험

  • 송신기 — 데이터를 보내는 장치
  • 수신기 — 데이터를 받는 장치
  • 전송매체 — 데이터가 이동하는 경로
  • 메시지 — 전달되는 데이터 내용
  • 프로토콜 — 통신 규칙의 집합

신호변환 장치

  • 모뎀 (Modem) : 디지털 데이터 ↔ 아날로그 신호 변환
  • DSU (Digital Service Unit) : 디지털 데이터 → 디지털 신호
  • CSU (Channel Service Unit) : 여러 채널을 다중화하여 전송 (T-1, E-1 방식)

통신제어장치 (CCP/FEP)

  • 대형 컴퓨터가 데이터 처리에만 전념할 수 있도록 데이터 전송을 대신 제어
  • 기능: 전송제어, 동기 및 오류제어, 기타 기능

통신회선 (전송매체)

  • 유선 : 트위스티드 페어, 동축케이블, 광케이블
  • 무선 : 전자파, 적외선

단말장치

  • 입출력 기능
  • 전송제어 기능
  • 기억 기능

네트워크의 필요성

  • 점대점(P2P) 연결 : 가장 간단한 형태, 그러나 통신 비용 측면에서 비경제적
  • 네트워크 연결 : 단말장치 1개의 포트만으로 노드에 접속, 설비 공유로 연결 비용 절감

데이터 통신의 역사

1837 전신기 (모르스) 최초의 전기통신
모르스가 전신기를 개발하여 최초로 전기를 이용한 통신 구현
1876 전화 (벨) 음성통신의 시초
알렉산더 그레이엄 벨이 전화를 발명
1895 무선전신 (마르코니)
마르코니가 무선전신기를 발명하여 선 없이 통신 가능
1946 ENIAC (에거트·모클리) 최초의 전자식 컴퓨터
에거트와 모클리가 최초의 전자식 컴퓨터 ENIAC 개발
1951 UNIVAC (에커트·모클리) 최초의 상업용 컴퓨터
유니박을 개발하여 미국 인구조사통계국에 설치
1958 SAGE (미국 공군) 최초의 데이터 통신 시스템
반자동 방공 시스템(Semi Automatic Ground Environment)
1963 SABRE (아메리칸 에어라인) 최초의 상업용 데이터 통신 시스템
항공사 좌석 예약 시스템으로 데이터 통신 상업화 시작
1964 CTSS (MIT) 최초의 시분할 시스템
대학 내 대형 컴퓨터의 공동 이용을 목적으로 개발
1969 ARPANET (미국 국방성) 최초의 패킷 통신망
미국 국방성이 개발한 최초의 패킷 통신망. 인터넷의 효시.
1970 ALOHA (하와이 대학) 최초의 무선 패킷 통신망
미국 하와이 대학에서 개발한 최초의 무선 패킷 통신망
1974 SNA (IBM)
IBM사가 개발한 컴퓨터 간 접속 표준
1975 TELENET, TYMNET
패킷 교환 방식의 데이터 통신 서비스 상용화 개시
1990 WWW (월드 와이드 웹)
최초의 하이퍼텍스트 브라우저 개발 및 하이퍼텍스트 등장

핵심 캐치

⚡
데이터(raw)와 정보(processed)는 다르다. 데이터를 가공해야 비로소 정보가 된다.
⚡
데이터 통신은 단순히 "데이터 보내기"가 아니라 처리(Processing) + 전송(Transmission)의 결합이다.
⚡
모든 통신에는 반드시 프로토콜(약속된 규칙)이 존재한다. 규칙 없이는 통신 불가.
⚡
역사에서 "최초" 타이틀에 집중하라. SAGE(최초 데이터 통신), SABRE(최초 상업용), ENIAC(최초 전자식 컴퓨터), CTSS(최초 시분할).
⚡
데이터 통신 네트워크 = 컴퓨터 네트워크 = 네트워크. 세 용어는 같은 개념이다.
⚡
데이터 통신 시스템의 5가지 구성요소는 시험 필수: 송신기, 수신기, 전송매체, 메시지, 프로토콜.
⚡
모뎀은 디지털↔아날로그 변환, DSU/CSU는 디지털↔디지털 변환. 변환 방향을 구분하라!
⚡
ARPANET(1969)은 최초의 패킷 통신망이자 인터넷의 효시. ALOHA(1970)는 최초의 무선 패킷 통신망.
⚡
무선 전송매체는 "공기"가 아니라 전자파·적외선이다. 혼동 금지.
⚡
점대점 연결은 단순하지만 비용이 많이 든다. 네트워크 연결은 설비를 공유하여 비용을 절감한다.

핵심 키워드

데이터 (Data)
현실 세계에서 관찰·측정을 통해 수집한 사실이나 값을 문자, 숫자, 기호 등으로 표현한 것. 아직 가공 전 원시 상태.
정보 (Information)
데이터를 가공·처리하여 가치와 의미를 부여한 것.
통신 (Communication)
두 사람(장치) 이상이 정해진 규칙(프로토콜)에 따라 한 지점에서 다른 지점으로 정보를 전달하는 과정.
데이터 통신 (Data Communication)
통신 기능을 가진 둘 이상의 장치 사이에서 전송매체를 통해 규칙에 따라 데이터로 표현된 정보를 교환하는 과정.
프로토콜 (Protocol)
통신 장치 간 데이터를 교환하기 위해 서로 약속한 규칙의 집합.
전송매체 (Transmission Medium)
데이터가 이동하는 물리적 경로. 유선(케이블)과 무선(전파) 모두 포함.
LAN (Local Area Network)
제한된 지역(건물, 캠퍼스) 내의 장치들을 연결하는 근거리 통신망.
WAN (Wide Area Network)
국가 또는 대륙 단위의 넓은 지역을 연결하는 광역 통신망.
시분할 시스템 (Time-Sharing System)
하나의 컴퓨터를 여러 사용자가 시간을 나눠 공동 이용하는 방식. CTSS가 최초.
모뎀 (Modem)
디지털 데이터를 아날로그 신호로 변환(변조)하거나, 아날로그 신호를 디지털 데이터로 역변환(복조)하는 신호변환 장치.
DSU / CSU
DSU(Digital Service Unit): 디지털 데이터를 디지털 신호로 변환. CSU(Channel Service Unit): 여러 채널을 다중화하여 전송(T-1, E-1 방식).
통신제어장치 (CCP / FEP)
대형 컴퓨터가 데이터 처리에만 전념할 수 있도록, 데이터 전송에 관한 전반적인 제어 기능을 대신 수행하는 장치. 전송제어·동기·오류제어 담당.
ARPANET
1969년 미국 국방성이 개발한 최초의 패킷 통신망. 오늘날 인터넷의 효시.
패킷 교환 (Packet Switching)
데이터를 일정 크기의 패킷으로 분할하여 전송하는 방식. ARPANET에서 처음 사용.
